# Control Flow
## IF
#### The if statement is used to execute a block of code only if a given condition evaluates to True. If the condition is False, the block is skipped.
## IF-ELSE
#### The if-else statement provides an alternative code block to execute when the condition in if evaluates to False. This ensures that either the if block or the else block runs, but not both.
## IF-ELIF-ELSE
#### The if-elif-else structure allows checking multiple conditions sequentially, where the first True condition's block executes. If none of the conditions are True, the else block (if present) runs as a fallback.
## Nested IFs
#### A nested if means placing one if statement inside another, allowing more complex decision-making. The inner if executes only if the outer if condition is True.

# Loops
#### Loops allow repeated execution of a block of code while a condition holds true or for a fixed number of iterations. Python primarily uses for and while loops for iteration.
## For Loop
#### A for loop iterates over a sequence (like a list, tuple, or string), executing the block of code for each element. It is commonly used with functions like range() for controlled iteration.
## While Loop
#### A while loop runs as long as a specified condition remains True, making it useful for indefinite iterations. It is typically used when the number of iterations is not known beforehand.

### Continue
#### The continue statement skips the current iteration of a loop and proceeds to the next iteration. It is useful for bypassing certain conditions without breaking the entire loop.
### Break
#### The break statement immediately exits the loop, regardless of the loop condition. It is often used to stop a loop early when a specific condition is met.


# Basic Library Usage
## Keyword library
#### The keyword module in Python allows checking for reserved words in the language. It provides functions like keyword.iskeyword() to determine whether a string is a Python keyword.
